# Interactive Stopwatch App

## Description
A simple and intuitive stopwatch application that allows users to track elapsed time, record lap times, and configure settings to suit their needs.

## Features
* Start, stop, and reset the timer
* Record lap times
* Display elapsed time
* Configurable settings

## How to Run
Open `index.html` in any modern browser (Chrome, Firefox, Safari).

## Usage
1. Open the app and click the start button to begin the timer.
2. Click the lap button to record lap times.
3. Click the stop button to pause the timer.
4. Click the reset button to reset the timer.

## Technologies Used
* HTML5
* CSS3 (Keyframe Animations)
* Vanilla JavaScript (ES6+)

## Project Structure
* `index.html`: The main entry point of the application.
* `styles.css`: Contains all CSS styles and keyframe animations.
* `script.js`: Contains all JavaScript logic and functionality.
* `README.md`: This file, containing project information and documentation.
